# Тестовое задание

Сделать страницу профиля пользователя с секцией "Финансы".
Форма редактирования параметров:
- выбор страны
- имя
- адрес

Секция финансов:
- баланс
- диалог пополнения с помощью карты (от себя добавил кнопку вывода средств с баланса)
- список последних транзакций(пополнения и списания содержат дату и сумму)

Использовать: Vue3, Pinia, Tailwind

### Запуск проекта
`npm install`

`npm run dev`
